# chidinma-va-portfolio
Chidinma_Chukwukpee_Portfolio
A modern, high-converting personal portfolio website designed for a Virtual Assistant. This project leverages a custom architectural split-screen layout to effectively showcase professional services, streamlined organizational workflows, and direct contact avenues for potential founder clients.

🚀 Features
Split-Screen Hero Architecture: Desktop-optimized layout utilizing CSS Flexbox to ensure a professional, high-impact first impression.

Custom Architectural Frame: Unique "Notched Corner" profile image styling created via CSS clip-path.

Interactive Skills Showcase: A responsive grid system featuring hover-state animations for tool and skill visualization.

Seamless Integration: Fully functional contact system powered by Formspree, alongside direct CTA buttons to project management platforms (Notion, Asana, Trello, etc.).

Mobile-First Design: Fully responsive CSS media queries that ensure the layout stacks and scales perfectly across mobile, tablet, and desktop devices.

🛠 Tech Stack
Frontend UI: HTML5, CSS3 (Flexbox, Grid, Custom clip-path), JavaScript (ES6).

Deployment: GitHub Pages.

Forms: Formspree API.

📦 How to View Locally
Clone or download this repository to your local machine.

Navigate to the project folder.

Open index.html directly in any modern web browser to view the site.
